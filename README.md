# Login Form
 
## Description
A login/register form to showcase my understanding of technologies and styling techniques. When you click 'Register' a new form will appear to create an account and if you already have an account you can click 'Sign up' to go back the the log  in page. Unfortunately you are not signing up for anything this is a mockup version!
